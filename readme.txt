== Description ==

**Reduce time giving support**

The LifeGuard+ Assistant plugin puts WordPress video tutorials right into the WordPress Dashboard. This is great for teaching clients how to use WordPress and reducing time spent giving support. It's a mutually beneficial arrangement - developers save time and clients can learn more and at their own pace.

**Increase Revenue from Theme Sales**

Install this plugin on client sites, or [use our code snippet for theme developers and authors](http://wplifeguard.com/lifeguard-plugin/lifeguard-assistant-for-developers/ "LifeGuard Assistant for Developers"), add your affiliate ID, and you can make an extra $8.70 on each theme you sell or give away. That could be a 58% increase in profits!

[More information about our affiliate program](http://wplifeguard.com/lifeguard-plugin/ "wpLifeGuard Affiliates").

[Be sure to check out our free WordPress video tutorials](http://wplifeguard.com/free-wordpress-tutorial-videos/ "WordPress Video Tutorials").


== Installation ==

1. Upload 'lifeguard_assistant' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. (Optional) To get affiliate credit, visit the "Help Settings" under the settings menu and add your wpLifeGuard affiliate ID
